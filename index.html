Deployment Note

Ready for GitHub Pages, with an optional Electron shell for desktop use.

This repo is intentionally portable: no hardcoded machine paths, all relative assets, and browser-safe backup/export behavior.
Steam Default Save Path%USERPROFILE%\AppData\LocalLow\Wishfully\Planet of Lana 2\*.sav
Game Pass / Microsoft Store Save Root%LOCALAPPDATA%\Packages\<Planet of Lana II package family>\SystemAppData\wgs\

Steam and Game Pass folders are shown as defaults only. The editor accepts dropped files/folders so it works across different package IDs and installations.

Use the included Electron wrapper when you want native file dialogs and direct backup writes next to the source save.
